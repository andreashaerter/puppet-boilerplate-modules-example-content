module: gimp

Manages GIMP (GNU Image Manipulation Program), an advanced, free Open Source raster graphics editor.

Please note that the handling of important components is split into different module parts/classes. For instance:

  • GIMP main application: gimp (init.pp)
  • GIMP Plug-Ins: gimp::plugin (plugin directory)

Have a look at their documentation for usage examples. However, some starting points:

  • Install the main application and the Resynthesizer Plug-In: 
      class { 'gimp': }
  class { 'gimp::plugin::resynthesizer': }
  • Remove GIMP: 
      class { 'gimp': ensure => 'absent' }
